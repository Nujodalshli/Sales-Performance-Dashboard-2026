# Sales-Performance-Dashboard-2026
Interactive Power BI dashboard analyzing sales performance and trends across Saudi cities.
 📊 Sales Performance Dashboard 2026

### 📝 Project Overview
An interactive *Power BI* dashboard developed to analyze sales trends and performance across major cities in Saudi Arabia (Dammam, Jeddah, Riyadh).

### 🚀 Key Insights
* *Regional Analysis:* Detailed sales distribution by city.
* *Product Performance:* Tracking top-selling electronics.
* *Data-Driven Decisions:* Transforming raw Excel data into actionable visual insights.

### 🛠️ Tools Used
* *Power BI Desktop* (.pbix)
* *Microsoft Excel* (Data Source)
